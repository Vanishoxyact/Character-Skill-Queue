This mod adapts the Character Skill Queue mod (found [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2093326036]here[/url]) to work when using mods which give two skill points per level up. Note that this mod does nothing on its own, and the main Character Skill Queue mod is required.
